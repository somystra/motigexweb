# Motigex - Your Social Space
Created by NetGlobal Team. Inspired by Magic OS design.

## Project Tech Stack
- **Frontend:** Next.js (App Router), Tailwind CSS
- **UI Components:** Shadcn UI, Radix UI
- **Backend:** Supabase (Auth & Database)
- **Deployment:** Optimized for Netlify and Vercel

## Deployment Instructions
To host this project on Netlify without errors:
1. Ensure Environment Variables are set (`NEXT_PUBLIC_SUPABASE_URL`, `NEXT_PUBLIC_SUPABASE_ANON_KEY`).
2. Build command: `npm run build`
3. Publish directory: `.next` (or `dist` for Vite-based builds)
